# Archive

Chat history backups and deprecated content.

## Structure

- `chat_index.md` — Optional index of past conversations (populate manually, or via a custom skill on claude.ai where chat history is available)
- `HANDOFF_*.md` — Session handoff notes (created during major transitions)

## Retention

- Chat index entries: keep indefinitely (lightweight)
- Handoff files: keep 3 months, then summarize and delete
- Deprecated hub content: keep 1 month after deprecation
