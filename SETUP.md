# Setup

Pick your path:

- **Claude Code, git-only (default, ~10 min)** → [QUICKSTART.md](QUICKSTART.md). A
  private GitHub repo and the Claude Code you already run. No token in any file, no
  other service.
- **Guided from nothing (Claude walks you through it)** → paste the prompt in
  [START_HERE.md](START_HERE.md) or [FIRST_TIME_PROMPT.md](FIRST_TIME_PROMPT.md).
- **Optional cross-surface (claude.ai chat / mobile / automation)** → the MCP
  worker, [SETUP_MCP.md](SETUP_MCP.md). Advanced, later, needs a free Cloudflare
  account. Not required for Claude Code.

After setup, customize the *Route by topic* table in `CLAUDE.md`, your `hubs/`, and
`STATUS_SNAPSHOT.md`. See [ARCHITECTURE.md](ARCHITECTURE.md) for how it works and
[SECURITY.md](SECURITY.md) for what not to store.
